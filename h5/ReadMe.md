何謂迭代法
就是利用迴圈 不斷地更新變數的狀態，直到滿足特定的條件為止。每一次的重複執行，就稱為一次「迭代 (Iteration)」。

核心要素
一個成功的迭代法通常包含三個關鍵要素：

    1.初始狀態 (Initialization)： 設定變數的起始值。
    2.重複步驟 (Step/Loop)： 根據目前的狀態計算出新的狀態（舊的輸出變成新的輸入）。
    3.終止條件 (Termination Condition)： 什麼時候該停下來？通常是達到預定的次數，或是計算結果已經足夠精確。

與遞迴法的不同

迭代是自己一步一步做到底。 透過迴圈完成

遞迴是把任務切小，發包給分身去做，最後再匯總。 透過呼叫自己將問題拆成小問題
